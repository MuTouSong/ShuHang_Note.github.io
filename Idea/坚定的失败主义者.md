# 思想
1. 我们做的所有事情，最终都会失败
2. 坚定的失败主义者，绝不是投降主义者
3. 失败主义者是发现此路不通之后，调转方向，绞尽脑汁开拓出一条新的道路；而投降主义者是发现此路不通之后，就原地不动，坐以待毙，等待敌人大发慈悲
4. 失败主义者应该撇弃一切杂念，绞尽脑汁的解决问题，以便创造更多的失败可能性，远离更糟的失败
5. 失败不是立即的，在真正的、更糟失败到达之前，将失败的结局导向更加好一点的失败
6. 失败不只是一种结果，是一个循序渐进的过程，结果终将失败，应该更加重视失败的过程
